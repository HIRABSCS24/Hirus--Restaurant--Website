##Hirus--Restaurant--Website
A complete restaurant website built with HTML, CSS &amp; JavaScript — featuring multi-page layout, menu tabs, gallery filter, reservation form &amp; scroll animations.

●Hirus Restaurant — HR
A complete multi-page restaurant website built from 
scratch using pure HTML, CSS & JavaScript — 
no frameworks, no libraries.


●Live Demo
https://hirabscs24.github.io/Hirus--Restaurant--Website/index.html


●Pages
▪︎Home — Hero section, About, Menu preview, 
  Gallery, Testimonials, Reservation form
▪︎Menu — 6 cuisine categories with tab switching
▪︎Gallery — Filter by category AND lightbox popup
▪︎Reservation — Booking form with full validation
Contact — Contact form, & map


●Features & Techniques Used
1)CSS
- CSS `position: relative` & `absolute` for layered 
  double image layout
- CSS Variables for consistent theming
- CSS Grid & Flexbox for all layouts
- Smooth hover effects on cards & buttons
- Fully responsive — mobile, tablet & desktop
- Custom scrollbar styling

2)JavaScript
- Scroll reveal animations using 
  `IntersectionObserver` API
- Dynamic menu tab switching
- Gallery filter by category
- Image lightbox on click (ESC to close)
- Form validation with custom toast notifications
- Mobile hamburger menu toggle
- Reservation date validation 
  (past dates blocked)

3)Layout Highlights
- Layered double image with floating corner image
  and PKT. 2018 badge — pure CSS positioning
- Sticky navbar that darkens on scroll
- Hero section with gradient overlay
- Masonry-style gallery grid

●Built With
- HTML5
- CSS3
- Vanilla JavaScript
